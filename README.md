# Malaria Detection Project

This project focuses on detecting malaria from microscopic blood cell images using Deep Learning techniques. A Convolutional Neural Network (CNN) model is trained to classify whether a given cell image is Parasitized (Infected) or Uninfected (Healthy).

The goal is to automate the malaria diagnosis process, making it faster, more accurate, and accessible, especially in areas with limited medical resources.

## Files
- malaria_detection.py
- malaria_model.h5
- dataset (cell_images)
